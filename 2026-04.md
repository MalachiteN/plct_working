在调研过程中分析了 VS Code 源码对语言包安装、更新、卸载的特殊处理，并进行相应实验，确认了 onDidChange 事件不会在开发者未更新 package.json 中的 version 字段的情况下重新构建插件 vsix 并安装时发生，也确认了在 version 字段变化时 VS Code 自带的通知就会包括重载窗口按钮。从而确认该需求无法完成，进行合理论证，协助关闭对应的需求 Issue。
